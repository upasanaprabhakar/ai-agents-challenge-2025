## Getting Set Up

### What You'll Need
- Python 3.9+
- A Kaggle account
- Google Gemini API key
- Basic Python knowledge

### Quick Start

```bash
# Clone this repo
git clone https://github.com/upasanaprabhakar/ai-agents-challenge-2025.git
cd ai-agents-challenge-2025

# Install what's needed
pip install -r config/dependencies.txt

# Set up your API key
echo "GOOGLE_API_KEY=your_key_here" > .env
```

Check `config/setup-guide.md` for detailed instructions.
